2.  **ADB**: The Android Debug Bridge (`adb`) tool is located at:

3.  **Java/JDK**: This project requires a compatible JDK (Java 17 or 21).

To compile the application and generate the APK:

```bash

cd /Users/praxsmac/AndroidStudioProjects/MyApplication

# Build the Debug APK
./gradlew assembleDebug
```

`app/build/outputs/apk/debug/app-debug.apk`

## Running on a Device

1.  **Connect Device**: Ensure your Android device is connected via USB and USB Debugging is enabled.
2.  **Check Connection**:
    ```bash
    /Users/praxsmac/Library/Android/sdk/platform-tools/adb devices
    ```
    *You should see your device ID in the list.*

3.  **Install the APK**:
    ```bash
    /Users/praxsmac/Library/Android/sdk/platform-tools/adb install -r app/build/outputs/apk/debug/app-debug.apk
    ```

4.  **Launch the App**:
    The app package name is `com.example.myapplication`.
    ```bash
    /Users/praxsmac/Library/Android/sdk/platform-tools/adb shell am start -n com.example.myapplication/.MainActivity
    ```
    *(Note: Since this is a launcher integration, you might need to set it as default launcher manually via device settings, or launch the specific activity).*

## Troubleshooting

- **"Command not found: adb"**: Ensure you use the full path or add platform-tools to your PATH.
- **Build Failures**: Check `gradle.properties` ensures the correct JDK is used.
- **Install Failed**: Unlock the device screen and confirm the installation prompt if it appears.
